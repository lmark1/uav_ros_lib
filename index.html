Summary

A collection of useful libraries for the uav_ros_stack.

Documentation

Documentation can be found at lmark1.github.io/uav_ros_lib.

General

Namespaces

  • param_util - Useful functions for loading values from the ROS parameter server
  • nonlinear_filters - Implementation of simple, commonly used nonlinear filters
  • ros_convert - Useful functions for conversion to and from ROS messages, various attitude conversions etc.
  • trajectory_helper - Useful functions for generating simple trajectory references

Classes

  • Topic Handler - A helper class for handling ROS message subscriptions
  • Reconfigure Handler - A helper class for handling ROS reconfigure server
  • Global To Local - A class used for transforming [Lat,Lon,Alt] to and from ENU coordinates

Estimation

  • Kalman Filter - Vanilla - A straightforward implementation of a single measurement Discrete Kalman Filter
  • kf_base - A class providing a template for a generic kalman filter implementation
  • Constant Velocity LKF - An implementation of a constant velocity linear Kalman filter using the kf_base as the base class
  • Constant Acceleration LKF - An implementation of a constant acceleration linear Kalman filter using kf_base as the base class
  • Constant Acceleration UKF - An implementation of an Unscented Kalman Filter with a constant acceleration using kf_base as the base class
