Detailed Description

template<class T>
class ros_util::TopicHandlerWithFunction< T >

A Subscriber Topic Handler class used to setup the ROS subscriber handle and a timer to check if the topic is active. ROS Timer is only activated if a timeout is specified. Additionally it uses a given function to call during the subscriber callback.
